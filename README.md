# Worldline Tracker - 《杀戮尖塔2》SL辅助模组

## 项目简介

Worldline Tracker 是一个为《杀戮尖塔2》设计的辅助模组，旨在帮助玩家在SL（Save/Load）过程中记录和复现战斗动作。通过记录完整的战斗序列并在SL后提供提示，玩家可以更轻松地尝试不同的策略，寻找最优打法。

## 核心功能

### 动作记录
- **完整动作序列**：记录出牌、用药、遗物触发等所有战斗动作
- **详细上下文**：包括回合数、发起者、目标、是否为被动触发
- **实时记录**：在动作发生时立即记录，确保准确性

### SL恢复辅助
- **自动检测SL**：识别游戏重启/重载事件
- **历史提示**：SL后自动显示之前的动作序列
- **进度对比**：高亮建议的下一步动作，对比不同尝试

### 交互式UI
- **侧边栏面板**：屏幕右侧可展开/收起的动作列表
- **树形历史视图**：使用节点+连线表示动作，支持树形结构
- **交互节点**：可点击查看详情，可拖动重新排列
- **实时更新**：战斗过程中UI实时刷新

### 数据管理
- **独立存档**：每场战斗生成独立的JSON文件
- **文件浏览器**：查看、管理历史战斗记录
- **导出功能**：支持导出为可读格式

## 安装与使用

### 安装要求
- 《杀戮尖塔2》游戏本体（v0.99.1+）
- 支持C# mod的游戏版本
- **无需BaseLib** - 直接使用游戏原生API，简化安装流程

### 安装步骤
1. 将编译后的mod文件放入游戏mods目录
2. 在游戏模组管理界面启用Worldline Tracker
3. **无需额外依赖** - 模组直接使用游戏原生API，无需安装BaseLib

### 基本使用
1. **自动记录** - 开始战斗后，模组自动开始记录所有动作
2. **实时保存** - 战斗结束后，记录自动保存到JSON文件
3. **文件位置** - 战斗记录保存在 `用户数据目录/ModData/WorldlineTracker/Battles/`
4. **数据格式** - 标准JSON格式，包含回合数、动作类型、发起者、目标等完整信息
